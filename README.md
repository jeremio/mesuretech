# Mesure Tech

Une application SPA moderne créée avec Vue.js 3, Vite, TypeScript et Tailwind CSS 4.

## Technologies

- Vue.js 3 avec Composition API
- TypeScript
- Vite.js
- Vue Router
- Vue I18n (internationalisation)
- Tailwind CSS 4
- ESLint avec la configuration d'Anthony Fu
- PNPM comme gestionnaire de paquets

## Prérequis

- Node.js (v16+)
- PNPM

## Installation

```bash
# Installer les dépendances
pnpm install
```

## Développement

```bash
# Démarrer le serveur de développement
pnpm dev
```

## Production

```bash
# Compiler pour la production
pnpm build

# Prévisualiser la build de production
pnpm preview
```

## Linting

```bash
# Lancer ESLint
pnpm lint
```

## Déploiement

Ce projet est configuré pour être déployé sur Netlify.
